# BookMyStay - Hotel & Room Booking Platform

Welcome to the **BookMyStay** project repository.

## Project Overview
BookMyStay is a full-stack hotel and room booking platform designed to replicate features of major platforms like Booking.com or Airbnb. It supports three distinct user roles: Customers, Hotel Owners, and Administrators.
